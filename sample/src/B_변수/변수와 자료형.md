# 📦 02. 변수(Variable)와 자료형(Data Type)

프로그램이 돌아가려면 데이터를 어딘가에 임시로 보관해야 합니다. 이번 시간에는 데이터를 담는 바구니인 **변수**와 그 바구니의 종류인 **자료형**에 대해 알아보겠습니다.

---

## 1. 🏷️ 변수(Variable)란 무엇일까요?

변수는 **'데이터를 저장하기 위해 이름을 붙인 메모리 공간'**입니다. 

쉽게 비유하자면, 거대한 창고(메모리) 안에 수많은 상자가 있는데, 특정 상자에 **'나이'**라는 이름표를 붙이고 그 안에 숫자 **'25'**를 담아두는 것과 같습니다. 나중에 '나이'라고 부르기만 하면 언제든 그 안의 값을 꺼내 쓰거나 바꿀 수 있습니다.

---

## 2. 📝 변수 사용하기: 선언과 할당

자바에서 변수를 사용하려면 두 단계의 과정이 필요합니다.

1. **선언 (Declaration):** "컴퓨터야, 정수를 담을 'age'라는 이름의 상자를 하나 만들어줘!"
2. **할당 (Assignment):** "그 상자에 '25'라는 값을 넣어줘!"

(코드 예시)
int age;      // 1. 선언: 자료형(int)과 변수명(age) 결정
age = 25;     // 2. 할당: 대입 연산자(=)를 사용하여 값 저장

int score = 100; // 선언과 할당을 한 번에 할 수도 있습니다.

---

## 3. 🍱 자료형(Data Type): 상자의 종류

자바는 매우 꼼꼼한 언어라, 상자를 만들기 전에 어떤 종류의 물건을 담을지 미리 정해야 합니다. 이를 자료형이라고 합니다.

### ① 대표적인 기본 자료형 (Primitive Type)

| 종류 | 자료형 | 표현 범위 및 특징 | 예시 |
| :--- | :--- | :--- | :--- |
| **정수** | int | 가장 많이 쓰이는 정수 타입 (약 -21억 ~ 21억) | 10, -500 |
| **정수(큰값)** | long | int보다 훨씬 큰 정수 (끝에 L을 붙임) | 10000000000L |
| **실수** | double | 소수점이 있는 숫자 (정밀도가 높음) | 3.14, 0.5 |
| **논리** | boolean | 참(true) 또는 거짓(false)만 저장 | true, false |
| **문자** | char | 딱 한 글자만 저장 (작은따옴표 사용) | 'A', '가' |

### ② 문자열 (String)
String은 기본 자료형은 아니지만, 글자 여러 개를 담을 때 필수적으로 사용합니다. 큰따옴표(" ")를 사용해야 합니다.
예시: String name = "자바 학습자";

---

## 4. 📏 변수 이름 짓기 (Naming Convention)

컴퓨터는 a, b, c라고 지어도 이해하지만, 좋은 개발자는 누구나 이해할 수 있는 이름을 짓습니다.

1. **의미 있는 이름 사용**: int a; (X) -> int userAge; (O)
2. **카멜 케이스 (Camel Case)**: 여러 단어를 조합할 때 두 번째 단어부터 첫 글자를 대문자로 씁니다.
   예시: phoneNumber, studentScore, isLogin
3. **숫자로 시작 불가**: 1stPlace (X) -> place1 (O)
4. **대소문자 구분**: age와 Age는 서로 다른 변수입니다.

---

## 5. ⚠️ 주의할 점

* **변수는 한 번에 하나의 값만**: 새로운 값을 넣으면 기존에 있던 값은 사라집니다.
* **타입 일치의 법칙**: int 상자에는 정수만 넣어야 합니다. 실수나 문자열을 넣으려고 하면 자바가 에러를 냅니다.

---

> 💡 **입문자를 위한 Tip**
> 처음에는 자료형이 많아 보이지만, 대부분 **int, double, boolean, String** 이 네 가지만 사용하게 됩니다. 너무 부담 갖지 말고 이 네 가지부터 익숙해져 보세요!
