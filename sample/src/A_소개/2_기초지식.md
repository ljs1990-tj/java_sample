# 🛠️ 00. 자바 실습을 위한 기초 지식 (Quick Start)

본격적인 코딩 공부에 앞서, 자바 소스 코드를 구성하는 기본적인 요소들과 실습 환경(이클립스)에서 꼭 알아야 할 팁들을 정리해 보겠습니다.

---

## 1. 💬 코드의 설명서: 주석(Comment)

주석은 컴퓨터는 읽지 않고 **사람만 읽는 메모**입니다. 내가 작성한 코드를 나중에 다시 보거나, 동료에게 설명할 때 사용합니다.

- **한 줄 주석 ( // )**: 슬래시 두 개 뒤의 내용은 모두 주석 처리됩니다.
- **여러 줄 주석 ( /* ... */ )**: 여러 줄에 걸쳐 긴 설명을 적을 때 사용합니다.

(주석 예시)
// 이것은 한 줄 주석입니다. 변수의 의미를 적을 때 좋아요.

/*
   이것은 여러 줄 주석입니다.
   프로그램 전체의 목적이나 복잡한 로직을 설명할 때 사용합니다.
*/

---

## 2. 📢 화면에 말하기: 출력문 (Print)

프로그램이 잘 작동하는지 확인하는 가장 기본적이고 확실한 방법은 화면에 값을 출력해 보는 것입니다.

- **System.out.println()**: 괄호 안의 내용을 출력하고 **줄을 바꿉니다.** (가장 많이 사용!)
- **System.out.print()**: 내용을 출력하지만 **줄을 바꾸지 않고** 이어서 출력합니다.

> **💡 이클립스 꿀팁:**
> `sysout` 이라고 입력한 뒤 **[Ctrl] + [Space]**를 누르면 `System.out.println();` 코드가 자동으로 완성됩니다!

---

## 3. ⌨️ 이클립스(Eclipse) 필수 단축키

코딩의 속도를 높여주고 실수를 줄여주는 필수 단축키들입니다.

| 단축키 | 기능 | 설명 |
| :--- | :--- | :--- |
| **Ctrl + Space** | **자동 완성** | 클래스 이름이나 명령어를 입력할 때 추천 목록을 보여줍니다. |
| **Ctrl + F11** | **실행 (Run)** | 작성한 코드를 컴파일하고 즉시 실행합니다. |
| **Ctrl + S** | **저장 (Save)** | 수정한 내용을 저장합니다. (저장해야 실행 결과에 반영됩니다) |
| **Ctrl + Shift + F** | **코드 정렬** | 들여쓰기가 흐트러진 코드를 깔끔하게 자동 정렬해줍니다. |
| **Ctrl + / ** | **주석 토글** | 선택한 줄을 즉시 주석으로 만들거나 일반 코드로 되돌립니다. |
| **Ctrl + D** | **줄 삭제** | 현재 커서가 위치한 줄을 통째로 삭제합니다. |

---

## 4. 🏗️ 자바 파일의 기본 구조 이해하기

모든 자바 소스 파일은 다음과 같은 약속된 틀을 가지고 있어야 합니다.

(구조 예시)
public class MyFirstProgram { // 1. 클래스: 파일 이름(MyFirstProgram.java)과 반드시 같아야 합니다.

    public static void main(String[] args) { // 2. 메인 메서드: 프로그램이 시작되는 '입구'입니다.
        
        // 3. 이곳에 우리가 시킬 명령들을 작성합니다.
        System.out.println("자바 공부를 시작합니다!"); 

    }
}

- **세미콜론 ( ; )**: 자바에서 문장의 끝에는 반드시 `;`를 찍어야 합니다. 마침표와 같은 역할입니다.
- **대소문자 구분**: 자바는 대문자와 소문자를 완전히 다른 글자로 취급합니다. (Main과 main은 다릅니다!)

---

## 5. 🚀 시작하기 전 체크리스트

1. **에러(빨간 줄)를 두려워 마세요**: 이클립스에서 빨간 줄 위에 마우스를 올리면 해결 방법(Quick Fix)을 제시해줍니다.
2. **파일 이름 주의**: 클래스 이름과 파일 이름이 다르면 프로그램이 돌아가지 않습니다.
3. **들여쓰기의 중요성**: 코드가 어떤 괄호 `{ }` 안에 들어있는지 잘 보여야 에러를 찾기 쉽습니다. (Ctrl+Shift+F를 자주 쓰세요!)
