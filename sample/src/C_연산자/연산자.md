# ➕ 03. 연산자 (Operators)

프로그램은 단순히 데이터를 저장하는 것에 그치지 않고, 그 데이터를 더하고, 빼고, 비교하며 새로운 결과를 만들어냅니다. 이때 사용하는 기호들을 **연산자**라고 부릅니다.

---

## 1. 🔢 산술 연산자 (Arithmetic Operators)

가장 기본이 되는 수학 계산 도구입니다.

- **`+` (더하기)**: 두 값을 더함
- **`-` (빼기)**: 앞의 값에서 뒤의 값을 뺌
- **`*` (곱하기)**: 두 값을 곱함
- **`/` (나누기)**: 앞의 값을 뒤의 값으로 나눔 (정수끼리 나누면 결과도 정수!)
- **`%` (나머지)**: 나누고 남은 **나머지 값**을 구함 (매우 중요!)



> **💡 나머지 연산자(%)는 어디에 쓸까요?**
> 홀수/짝수를 구분하거나(2로 나눈 나머지가 0이면 짝수), 배수를 찾을 때 아주 유용하게 사용됩니다.

---

## 2. 📝 대입 및 복합 대입 연산자 (Assignment Operators)

변수에 값을 저장할 때 사용하며, 계산과 대입을 동시에 할 수도 있습니다.

- **`=`**: 오른쪽의 값을 왼쪽 변수에 저장 (같다는 뜻이 아님에 주의!)
- **`+=`**: 더한 후 저장 (예: a += b  →  a = a + b)
- **`-=`**: 뺀 후 저장
- **`*=`**: 곱한 후 저장
- **`/=`**: 나눈 후 저장

(코드 예시)
int num = 10;
num += 5; // num은 15가 됩니다.
num *= 2; // num은 30이 됩니다.

---

## 3. ⚖️ 비교 연산자 (Comparison Operators)

두 값을 비교하여 결과로 **참(true)** 또는 **거짓(false)**을 내놓습니다. 나중에 배울 '조건문'에서 핵심적인 역할을 합니다.

- **`==`**: 두 값이 같으면 true (등호가 2개임에 주의!)
- **`!=`**: 두 값이 다르면 true
- **`>` / `<`**: 크다 / 작다
- **`>=` / `<=`**: 크거나 같다 / 작거나 같다

(비교 예시)
5 == 5  // true
5 != 3  // true
10 < 5  // false

---

## 4. 🧠 논리 연산자 (Logical Operators)

여러 개의 조건을 조합할 때 사용합니다.

- **`&&` (AND, 그리고)**: 두 조건이 **모두 true**여야 결과가 true입니다.
- **`||` (OR, 또는)**: 두 조건 중 **하나만 true**여도 결과가 true입니다.
- **`!` (NOT, 부정)**: 결과를 반대로 뒤집습니다. (true -> false)



---

## 5. 🆙 증감 연산자 (Increment & Decrement)

변수의 값을 1씩 증가시키거나 감소시킬 때 사용합니다. 위치에 따라 결과가 달라지니 주의해야 합니다.

- **`++num` (전위)**: 먼저 1을 더하고 나서 코드를 실행합니다.
- **`num++` (후위)**: 코드를 먼저 실행하고 나서 나중에 1을 더합니다.

(증감 예시)
int a = 10;
System.out.println(++a); // 11 출력 (먼저 더함)

int b = 10;
System.out.println(b++); // 10 출력 (출력 후 더함)
System.out.println(b);   // 11 출력 (이제서야 더해진 것을 확인)

---

## 6. ⚠️ 입문자가 자주 하는 실수

1. **`=`와 `==`의 혼동**: `=`는 값을 넣는 것이고, `==`는 같은지 물어보는 것입니다.
2. **나눗셈 주의**: 정수 5를 2로 나누면(`5 / 2`) 결과는 2.5가 아니라 **2**가 됩니다. (자바에서 정수 연산의 결과는 정수이기 때문입니다.) 소수점 결과를 얻으려면 `5.0 / 2`처럼 연산 값중 하나라도 실수형으로 연산해야 합니다.
