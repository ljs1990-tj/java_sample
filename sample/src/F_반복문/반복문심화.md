# 🔄 06. 반복문 심화: 중첩과 무한 루프 제어

기본적인 반복문을 넘어, 반복문을 겹쳐 사용하거나 의도적으로 무한히 반복시키는 고급 기법들을 배워봅시다. 이 내용들을 익히면 비로소 복잡한 프로그램을 만들 수 있게 됩니다.

---

## 1. 🍱 반복문 안의 반복문: 중첩 반복문 (Nested Loops)

반복문 안에 또 다른 반복문을 넣는 것을 말합니다. 바깥쪽 반복문이 한 번 돌 때마다, 안쪽 반복문은 전체를 다 돕니다.

- **가장 대표적인 예**: 구구단 출력, 2차원 표 그리기 등

(이중 for 문 구조 이해)
for (int i = 1; i <= 3; i++) {       // 바깥쪽 (행)
    for (int j = 1; j <= 5; j++) {   // 안쪽 (열)
        // 이 안의 코드는 총 3 * 5 = 15번 실행됩니다.
    }
}



---

## 2. ♾️ 의도적인 무한 루프와 탈출 (Infinite Loops & break)

조건식에 `true`를 넣으면 프로그램은 멈추지 않고 영원히 반복됩니다. 보통 사용자가 "종료"를 선택하기 전까지 계속 돌아가는 메뉴 시스템을 만들 때 사용합니다.

- **핵심**: 반드시 내부에 특정 조건이 맞으면 멈추게 하는 `break;`가 있어야 합니다!

(무한 루프 구조)
while (true) {
    System.out.println("명령을 입력하세요 (0: 종료)");
    int input = sc.nextInt();

    if (input == 0) {
        System.out.println("프로그램을 종료합니다.");
        break; // 루프를 즉시 빠져나갑니다.
    }
}

---

## 3. ⏩ 이번 판은 건너뛰자: continue 문

`break`가 반복문을 아예 끝내는 것이라면, `continue`는 **"이번 회차의 남은 코드만 건너뛰고 다음 회차로 바로 가자"**는 뜻입니다.

(continue 예시: 홀수만 출력하기)
for (int i = 1; i <= 10; i++) {
    if (i % 2 == 0) { 
        continue; // 짝수라면 아래 print를 무시하고 위로 올라가 i++를 수행합니다.
    }
    System.out.println(i); // 결과: 1, 3, 5, 7, 9
}

---

## 4. 🏁 이름 붙은 반복문 (Labeled Break) - 심화

중첩 반복문에서 안쪽 루프가 아니라 **바깥쪽 루프까지 한꺼번에 탈출**하고 싶을 때 사용합니다.

(라벨 예시)
ExitLabel: // 라벨 이름 정의
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
        if (i + j == 7) {
            break ExitLabel; // 모든 반복문을 한 번에 종료!
        }
    }
}

---

## 5. ⚠️ 심화 반복문 작성 팁

1. **변수 이름 주의**: 이중 루프를 쓸 때 바깥쪽은 `i`, 안쪽은 `j`나 `k`를 쓰는 것이 관례입니다. 하지만 더 의미 있는 이름(row, col 등)을 쓰면 코드가 훨씬 잘 읽힙니다.
2. **무한 루프의 위험**: `break` 조건이 절대로 성립될 수 없는 로직이라면 프로그램이 컴퓨터 자원을 모두 소모하며 강제 종료될 수 있습니다.
3. **가독성 유지**: 루프가 3단, 4단으로 겹쳐지면 사람이 이해하기 힘들어집니다. 그럴 땐 코드를 나누거나 설계가 잘못되지는 않았는지 검토해야 합니다.
