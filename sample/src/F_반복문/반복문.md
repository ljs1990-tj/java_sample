# 🔄 06. 반복문 (Loops)

프로그래밍의 가장 큰 장점은 지치지 않고 똑같은 일을 반복할 수 있다는 것입니다. 자바에서 제공하는 여러 가지 반복문을 통해 효율적인 코드를 작성하는 법을 배워봅시다.

---

## 1. 🔢 횟수가 정해져 있다면? for 문

`for` 문은 보통 반복할 횟수가 명확할 때 사용합니다. '초기화', '조건식', '증감식' 세 가지 요소가 한 줄에 들어 있어 가독성이 좋습니다.

(for 문 구조)
for (초기화; 조건식; 증감식) {
    // 조건이 참인 동안 반복할 코드
}

(예시: 1부터 5까지 출력하기)
for (int i = 1; i <= 5; i++) {
    System.out.println("현재 숫자는: " + i);
}

---

## 2. 🌀 조건이 맞을 때까지! while 문

`while` 문은 특정 조건이 참(true)인 동안 계속해서 반복합니다. 반복 횟수가 불명확할 때 주로 사용합니다.

(while 문 구조)
while (조건식) {
    // 조건이 참인 동안 반복할 코드
}

(예시: 숫자가 10보다 작을 때까지 반복)
int count = 1;
while (count <= 10) {
    System.out.println("카운트: " + count);
    count++; // [주의] 증감식을 빠뜨리면 무한 루프에 빠집니다!
}

---

## 3. 🦶 일단 한 번은 실행해! do-while 문

`while` 문과 비슷하지만, **조건을 나중에 검사**한다는 차이가 있습니다. 즉, 조건이 틀리더라도 최소한 한 번은 무조건 실행됩니다.

(do-while 예시)
int i = 100;
do {
    System.out.println("조건이 틀려도 한 번은 출력됩니다.");
} while (i < 10);

---

## 4. 🛑 반복 제어하기: break & continue

때로는 반복문을 중간에 멈추거나 특정 회차를 건너뛰어야 할 때가 있습니다.

- **break**: 반복문을 즉시 **종료**하고 빠져나갑니다.
- **continue**: 현재 반복 회차를 중단하고 **다음 반복(증감식/조건식)**으로 바로 넘어갑니다.

(예시: 3이 되면 탈출하기)
for (int i = 1; i <= 10; i++) {
    if (i == 3) break;
    System.out.println(i); // 1, 2만 출력되고 종료됩니다.
}

---

## 5. ⚠️ 반복문 사용 시 주의할 점

1. **무한 루프(Infinite Loop)**: 조건이 항상 참이 되어 프로그램이 멈추지 않는 현상입니다. 반복문을 탈출할 수 있는 장치(증감식 등)가 있는지 꼭 확인하세요.
2. **인덱스 범위**: `for` 문에서 사용하는 변수(주로 `i`)가 조건 범위를 벗어나지 않는지 주의해야 합니다.
3. **가독성**: 반복문이 너무 깊게 중첩(이중 for 문 등)되면 코드를 이해하기 어려워집니다.
