# 🚂 07-1. 1차원 배열 (1D Array) 기초 가이드

배열은 똑같은 종류의 변수들을 기차 칸처럼 일렬로 이어 붙인 것입니다. 변수를 수십 개 만들지 않고도 하나의 이름으로 관리할 수 있는 마법 같은 도구입니다.

---

## 1. 배열의 정의: "왜 배열을 쓸까요?"

우리가 학생 3명의 성적을 저장하려면 변수 3개가 필요합니다.
- int score1 = 90;
- int score2 = 80;
- int score3 = 70;

하지만 학생이 100명이라면 변수를 100개 만들 수 있을까요? 
배열을 사용하면 **`int[] scores = new int[100];`** 단 한 줄로 100개의 공간을 만들 수 있습니다.



---

## 2. 배열을 만드는 법 (생성 및 초기화)

배열은 "어떤 타입의 데이터를 담을 것인가?"와 "몇 칸을 만들 것인가?"가 핵심입니다.

### ① 선언과 동시에 공간 만들기 (빈 방 만들기)
방만 먼저 만들면, 안에는 기본값(숫자는 0)이 채워집니다.
- 문법: 자료형[] 배열이름 = new 자료형[칸수];
- 예시: int[] age = new int[3]; // 정수 3개를 담을 공간 확보

### ② 선언과 동시에 값 채우기 (이미 값을 알 때)
공간을 만듦과 동시에 원하는 데이터를 바로 집어넣습니다.
- 예시: int[] age = {20, 25, 30}; // 3칸짜리 배열이 만들어지고 값이 채워짐

---

## 3. 배열에 접근하는 법 (직접 접근)

배열의 가장 중요한 규칙은 **"방 번호(인덱스)는 0부터 시작한다"**는 것입니다.

### ① 인덱스(Index)의 원리
3칸짜리 배열 `int[] arr = new int[3];`이 있다면:
- 첫 번째 칸: arr[0]
- 두 번째 칸: arr[1]
- 세 번째 칸: arr[2]
- **주의**: arr[3]은 존재하지 않습니다! (0부터 세기 때문)



### ② 값 읽기와 쓰기 예시
// 값을 저장하는 법 (쓰기)
arr[0] = 10; // 0번 방에 10을 넣음
arr[1] = 20; // 1번 방에 20을 넣음

// 값을 가져오는 법 (읽기)
System.out.println(arr[0]); // 출력 결과: 10
int sum = arr[0] + arr[1]; // 10 + 20을 더해서 sum에 저장

---

## 4. 반복문으로 한꺼번에 접근하기

배열의 인덱스는 숫자가 0, 1, 2... 순서대로 늘어나기 때문에 `for`문과 환상의 짝꿍입니다.

(반복문 접근 예시)
int[] scores = {90, 80, 70, 60, 50};

for (int i = 0; i < scores.length; i++) {
    // i가 0, 1, 2, 3, 4로 변하면서 scores[i]에 순서대로 접근합니다.
    System.out.println((i+1) + "번째 점수: " + scores[i]);
}

---

## 5. 배열 사용 시 핵심 체크리스트

1. **타입 통일**: `int` 배열에는 `int`만 담을 수 있습니다. (`String`이나 `double` 혼합 불가)
2. **크기 고정**: 한 번 `3칸`으로 만든 배열은 중간에 `4칸`으로 늘릴 수 없습니다.
3. **0번의 법칙**: 모든 시작은 0번 인덱스부터라는 것을 절대 잊지 마세요!
