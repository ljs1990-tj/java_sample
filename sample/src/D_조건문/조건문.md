# 🚦 04. 조건문 (Conditional Statements)

지금까지의 프로그램은 위에서 아래로 순서대로만 실행되었습니다. 하지만 현실에서는 상황에 따라 다른 행동을 해야 할 때가 많죠. 이때 사용하는 것이 바로 **조건문**입니다.

---

## 1. 🤔 조건문이란 무엇일까요?

조건문은 **"만약 ~라면 A를 하고, 아니면 B를 해라"**와 같이 프로그램에 판단력을 심어주는 도구입니다.

- 예: 만약 점수가 60점 이상이면 "합격", 아니면 "불합격" 출력.
- 예: 로그인 상태가 참(true)이면 "마이페이지", 아니면 "로그인 창" 표시.

---

## 2. ☝️ 가장 기본: if 문

가장 단순한 형태의 조건문입니다. 괄호 `( )` 안의 조건이 **참(true)**일 때만 중괄호 `{ }` 안의 코드를 실행합니다.

(if 문 구조)
if (조건식) {
    // 조건이 참일 때 실행할 코드
}

(예시)
int age = 20;
if (age >= 19) {
    System.out.println("성인입니다.");
}

---

## 3. ✌️ 두 갈래 길: if-else 문

조건이 참일 때와 거짓일 때 실행할 코드를 각각 정해줍니다.

(if-else 예시)
int score = 55;
if (score >= 60) {
    System.out.println("합격입니다!");
} else {
    System.out.println("불합격입니다. 다음에 다시 도전하세요.");
}

---

## 4. ✋ 여러 갈래 길: if-else if-else 문

확인해야 할 조건이 여러 개일 때 사용합니다. 위에서부터 순서대로 확인하며, 하나라도 참이 나오면 그 줄을 실행하고 전체 조건문을 빠져나옵니다.

(등급 나누기 예시)
int score = 85;

if (score >= 90) {
    System.out.println("A 등급");
} else if (score >= 80) {
    System.out.println("B 등급"); // 여기에서 걸리게 됩니다!
} else if (score >= 70) {
    System.out.println("C 등급");
} else {
    System.out.println("D 등급");
}

---

## 5. ⚡ 값이 딱 정해져 있다면? switch 문

비교하려는 값이 특정 정수나 문자열로 딱 떨어질 때 `if` 문보다 깔끔하게 작성할 수 있습니다.

- **case**: 값이 무엇인지 확인하는 지점.
- **break**: 해당 실행을 마치고 switch 문을 빠져나오게 하는 중요한 장치! (없으면 다음 case까지 실행됨)
- **default**: 어떤 case에도 해당하지 않을 때 실행 (if 문의 else와 같은 역할).

(요일 출력 예시)
int day = 3;
switch (day) {
    case 1:
        System.out.println("월요일");
        break;
    case 2:
        System.out.println("화요일");
        break;
    case 3:
        System.out.println("수요일"); // 실행됨
        break;
    default:
        System.out.println("알 수 없는 요일");
}

---

## 6. ⚠️ 조건문 작성 시 주의할 점!

1. **중괄호 { } 생략 금지**: 실행할 코드가 한 줄이면 중괄호를 안 써도 되지만, 입문자 때는 실수를 방지하기 위해 **무조건 쓰는 습관**을 들이는 것이 좋습니다.
2. **비교 연산자 사용**: 조건식 결과는 반드시 `true` 또는 `false`여야 합니다. (`if (score = 100)` 같은 실수는 금물! `=`는 대입, `==`가 비교입니다.)
3. **switch 문에서의 break**: `break`를 빼먹으면 원하는 결과가 나오지 않을 수 있으니 항상 체크하세요.
