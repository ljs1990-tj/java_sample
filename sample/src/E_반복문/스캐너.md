# ⌨️ 05. 입력받기: Scanner 클래스

지금까지의 프로그램은 우리가 미리 정해둔 변수 값만 계산했습니다. 하지만 진짜 프로그램은 **사용자가 입력하는 값**에 따라 결과가 달라져야 합니다. 자바에서 입력을 받는 가장 쉬운 방법인 `Scanner`를 배워봅시다.

---

## 1. 📋 Scanner 사용을 위한 3단계 준비

`Scanner`는 자바가 기본적으로 제공하는 도구상자(라이브러리) 안에 들어있습니다. 이를 꺼내 쓰려면 다음 세 단계가 필요합니다.

1. **도구 가져오기**: 코드 맨 윗줄에 `import java.util.Scanner;`를 적습니다.
2. **Scanner 만들기**: `Scanner sc = new Scanner(System.in);`을 입력하여 도구를 생성합니다.
3. **값 읽어오기**: 사용자가 키보드로 치는 값을 읽는 명령어를 사용합니다.

---

## 2. 📥 데이터 타입별 입력 메서드

입력받을 데이터가 숫자인지 문자인지에 따라 사용하는 명령어가 다릅니다.

- **sc.nextInt()**: 정수(int)를 입력받습니다.
- **sc.nextDouble()**: 실수(double)를 입력받습니다.
- **sc.next()**: 공백(스페이스) 전까지의 문자열을 입력받습니다.
- **sc.nextLine()**: 한 줄 전체(엔터 키를 치기 전까지)를 입력받습니다.

(사용 예시)
System.out.print("나이를 입력하세요: ");
int age = sc.nextInt();

System.out.print("이름을 입력하세요: ");
String name = sc.next();

---

## 3. ⚠️ Scanner 사용 시 주의할 점

### ① import 문 누락 주의
`Scanner`라는 글자에 빨간 줄이 뜬다면, 파일 가장 윗부분(package 아래)에 `import java.util.Scanner;`가 있는지 확인하세요. 이클립스 단축키 **[Ctrl] + [Shift] + [O]**를 누르면 자동으로 추가됩니다.

### ② nextInt()와 nextLine() 혼용 주의
`nextInt()`로 숫자를 입력받고 바로 다음에 `nextLine()`으로 문자열을 받으려고 하면, 숫자를 입력하고 친 '엔터(Enter)' 키가 `nextLine()`에 들어가버려 입력을 건너뛰게 되는 현상이 발생합니다.
- **해결책**: 숫자를 받은 후 `sc.nextLine();`을 한 번 더 써서 엔터를 비워주거나, 가급적 같은 계열의 메서드를 사용합니다.

### ③ 도구 닫기 (Option)
프로그램이 끝날 때 `sc.close();`를 해주면 컴퓨터 자원을 아낄 수 있습니다. (필수는 아니지만 권장사항입니다.)

---

## 4. 🏗️ Scanner의 기본 구조

(구조 예시)
import java.util.Scanner; // 1. 도구 불러오기

public class ScannerBasic {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in); // 2. 도구 생성

        System.out.print("메시지를 입력하세요: ");
        String input = sc.nextLine(); // 3. 입력 대기 및 저장

        System.out.println("입력하신 내용은: " + input);
        
        sc.close(); // 4. 도구 닫기
    }
}
